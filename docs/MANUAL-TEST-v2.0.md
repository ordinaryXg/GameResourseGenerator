## 缺陷 Backlog（按优先级）

### BUG-001 · Prefab 导入未关联材质与贴图 `P0`

| 字段 | 内容 |
|------|------|
| 复现 | 导出多节点 prefab → 工具栏/拖拽再导入 → 仅 prefab 被选中 |
| 预期 | 材质 `.mtl`、贴图 `.png` 一并解析并写入 `assetRegistry`，AssetRef 可预览 |
| 模块 | `prefab-importer.ts` · `usePrefabImport.ts` · `import-asset-resolver` |
| 单测 | `prefab-importer.test` 通过但可能未覆盖多文件 pick / 拖拽场景 |

### BUG-002 · Prefab 拖拽 drop 区域残留 `P1`

| 字段 | 内容 |
|------|------|
| 复现 | 拖 prefab 到界面 → drop 高亮框出现，松手或取消后**不消失** |
| 预期 | drop 态仅绑定在**指定区域**（如预览区或导入区）；离开/结束拖拽立即清除 |
| 模块 | `usePrefabImport.ts` · `App.tsx` 全局 dragover/drop |

### BUG-003 · 导出 Cocos 节点朝向不一致 `P1`

| 字段 | 内容 |
|------|------|
| 复现 | 编辑器预览 Transform 与 Cocos 场景挂载 prefab 后朝向不一致 |
| 预期 | rotation / euler 与 Cocos 坐标系 round-trip 一致 |
| 模块 | `export-pipeline.ts` · Transform 序列化 |

### BUG-004 · 导入贴图后资产浏览器不实时刷新 `P2`

| 字段 | 内容 |
|------|------|
| 复现 | 导入 PNG 落盘成功，网格无新缩略图，切换面板后才出现 |
| 预期 | `registerProjectAsset` 后立即触发 asset-store / 浏览器 re-render |
| 模块 | `useAssetImport.ts` · `asset-store.ts` · `AssetBrowserPanel` |

### UX-001 · AI 面板切换入口 `P2`

| 建议 | 将 AI 显示/隐藏 Toggle 放在**左栏**（层级树旁），与层级/属性工作流一致 |

### UX-002 · 预览区 FPS 计数 `P2`

| 建议 | 状态栏或预览角落实时显示 FPS，便于 P1/P2 性能验收 |
