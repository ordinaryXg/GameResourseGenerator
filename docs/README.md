## 产品简介

**FX Studio**（包名：`cocos-effect-generator`）是一款面向 Cocos Creator 3.8 的游戏粒子特效编辑器。v2.0 起定位为**通用特效编辑器优先**，支持多 Emitter 组合、资产引用管理与内置资产库；AI 作为可选辅助能力。
