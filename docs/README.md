## 产品简介

**FX Studio** 的终极目标是：

> 做一款最好用的特效制作软件，产出可直接导入 **Cocos Creator / Unity / Unreal** 使用的特效资源。

v2.0 以通用特效编辑器为核心：多 Emitter 组合、资产引用、实时预览与 Cocos 3.8 round-trip；Unity / Unreal 导出在路线图中。
